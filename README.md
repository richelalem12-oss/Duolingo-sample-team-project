Duolingo-Sample-Team-Project

A collaborative 4-member front-end project recreating key parts of the Duolingo website, including the Landing Page, Get Started Page, Flag Section, and other UI components.

Overview

This project rebuilds major Duolingo UI sections using modern, responsive front-end practices. It focuses on teamwork, clean structure, and real-world GitHub collaboration.
